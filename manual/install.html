Installing Masumi

Thank you for your interest in Masumi. Installing is quite easy and requires only two steps.

Step 1: Installing Java

Masumi is written in the Java programming language. In order to run Masumi, you must have the Java runtime libraries installed. These libraries are free to download and install. You simply have to go to the Java home page and click on the Free Java Download button. Just follow the instructions from there.

Step 2: Installing Masumi

Simply download the Masumi installer and follow the instructions. It will ask you where you want to install the executable. If you don't know, the default is fine.

Uninstalling Masumi

If you no longer wish use Masumi you can easily untall it from the Add/Remove Programs section of the Windows Control panel.
